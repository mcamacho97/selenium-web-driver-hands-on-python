# Selenium Web Driver - Hands On (Python)

Sample project for automated testing with Selenium WebDriver in Python.

## Description

Small example project containing a basic test suite for a login page, using Selenium to drive the browser and `pytest` to run the tests and produce HTML reports.

## Requirements

- Python 3.8+ installed
- A compatible WebDriver (ChromeDriver, geckodriver, etc.) available in the `PATH` or configured in `utils/driver_factory.py`
- Project dependencies (see Installation section)

## Installation

1. Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
```

2. Install dependencies (use `requirements.txt` if present; otherwise install `pytest` and `selenium`):

```bash
pip install -r requirements.txt || pip install pytest selenium
```

3. Make sure the WebDriver (for example `chromedriver`) is on your `PATH` or configure its path in `utils/driver_factory.py`.

## Project Structure

- `pages/` - Page Objects (`base_page.py`, `login_page.py`)
- `utils/` - Utilities (e.g. `driver_factory.py`)
- `tests/` - Test cases (`test_login.py`)
- `conftest.py` - pytest fixtures
- `pyproject.toml` - Project metadata
- `report.html` - HTML report generated by `pytest`

## Running the tests

Run the login test and generate an HTML report with:

```bash
pytest tests/test_login.py --html=report.html
```

When finished, open `report.html` in your browser to view the results.

## Notes & Tips

- If you use Chrome, download the `chromedriver` version that matches your Chrome browser.
- To run the entire test suite, simply run `pytest`.
- To run tests in headless mode, adjust the browser options in `utils/driver_factory.py`.
